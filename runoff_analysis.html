径流数据相似度分析与预测系统技术文档

一、系统概述

本系统是一套针对历史月尺度径流数据的自动化分析与预测工具，主要实现两大核心功能：

  1. 多指标加权综合相似度分析：从数值大小和变化趋势两个维度，全面评估历史年份与目标年份前期径流的相似程度
  2. 相似年修正径流预测：基于最优相似年份，结合动态修正系数，预测目标年份全年径流数据

系统采用模块化设计，分为两个独立执行单元：

  • 宽表生成单元：负责数据预处理和相似度计算
  • 长表转换与预测单元：负责格式转换、径流预测和结果输出

二、多指标加权综合相似度检测模型

2.1 模型设计思路

传统相似度分析通常使用单一指标（如相关系数或欧氏距离），容易出现"数值相似但趋势不同"或"趋势相似但数值差异大"的误判。本系统采用水文行业标准的7指标加权综合评估体系，从多个维度全面衡量相似性，避免单一指标的局限性。

2.2 7个核心相似度指标及权重

指标名称 权重 评估维度 物理意义
皮尔逊相关系数相似度 0.25 变化趋势 衡量两个年份径流变化的线性相关性，是最重要的趋势指标
余弦相似度 0.20 变化趋势 衡量两个年份径流向量的夹角大小，反映变化方向的一致性
欧氏距离相似度 0.15 数值大小 衡量两个年份径流数值的绝对差异
相对误差均值相似度(MAPE) 0.15 数值大小 衡量两个年份径流的平均相对偏差
曼哈顿距离相似度 0.10 数值大小 衡量两个年份径流数值的平均绝对差异
最大相对误差相似度(MaxRE) 0.10 数值大小 衡量两个年份径流的最大单月偏差
趋势一致性指数(TCI) 0.05 变化趋势 衡量两个年份逐月变化方向的一致性

2.3 各指标计算公式

2.3.1 皮尔逊相关系数相似度

衡量两个变量之间的线性相关程度，取值范围[-1,1]，转换为0-1区间：

\[ r = \frac{\sum_{i=1}^{n}(x_i-\bar{x})(y_i-\bar{y})}{\sqrt{\sum_{i=1}^{n}(x_i-\bar{x})^2}\sqrt{\sum_{i=1}^{n}(y_i-\bar{y})^2}} \] \[ \text{皮尔逊相似度} = \frac{r + 1}{2} \]
- \(x_i\)：目标年份第\(i\)个月的径流数据
- \(y_i\)：历史年份第\(i\)个月的径流数据
- \(n\)：参与计算的月份数（当前为4个月）

2.3.2 余弦相似度

衡量两个向量的夹角余弦值，取值范围[0,1]：

\[ \text{余弦相似度} = \frac{\sum_{i=1}^{n}x_iy_i}{\sqrt{\sum_{i=1}^{n}x_i^2}\sqrt{\sum_{i=1}^{n}y_i^2}} \]

2.3.3 欧氏距离相似度

衡量两个向量的空间距离，转换为0-1区间：

\[ d = \sqrt{\sum_{i=1}^{n}(x_i-y_i)^2} \] \[ \text{欧氏距离相似度} = 1 - \frac{d}{2 \times \|\vec{x}\|} \]
- \(\|\vec{x}\|\)：目标年份径流向量的模长

2.3.4 相对误差均值相似度(MAPE)

衡量平均相对偏差，转换为0-1区间：

\[ \text{MAPE} = \frac{1}{n}\sum_{i=1}^{n}\left|\frac{x_i-y_i}{x_i}\right| \times 100\% \] \[ \text{MAPE相似度} = \frac{1}{1 + \text{MAPE}} \]

2.3.5 曼哈顿距离相似度

衡量平均绝对偏差，转换为0-1区间：

\[ d = \sum_{i=1}^{n}|x_i-y_i| \] \[ \text{曼哈顿距离相似度} = 1 - \frac{d}{2 \times \sum_{i=1}^{n}|x_i|} \]

2.3.6 最大相对误差相似度(MaxRE)

衡量最大单月偏差，转换为0-1区间：

\[ \text{MaxRE} = \max_{1 \leq i \leq n}\left|\frac{x_i-y_i}{x_i}\right| \times 100\% \] \[ \text{MaxRE相似度} = \frac{1}{1 + \text{MaxRE}} \]

2.3.7 趋势一致性指数(TCI)

衡量逐月变化方向的一致性，取值范围[-1,1]，转换为0-1区间：

\[ \text{TCI} = \frac{1}{n-1}\sum_{i=1}^{n-1}\text{sign}((x_{i+1}-x_i)(y_{i+1}-y_i)) \] \[ \text{TCI相似度} = \frac{\text{TCI} + 1}{2} \]
- \(\text{sign}(z)\)：符号函数，\(z>0\)时为1，\(z<0\)时为-1，\(z=0\)时为0

2.4 综合相似度得分计算

  1. 指标标准化：对每个指标进行Min-Max标准化，转换为0-1区间，消除不同量纲的影响：

    \[ S'_{ij} = \frac{S_{ij} - \min(S_j)}{\max(S_j) - \min(S_j)} \]
    - \(S_{ij}\)：第\(i\)个年份第\(j\)个指标的原始值
    - \(\min(S_j)\)：第\(j\)个指标的最小值
    - \(\max(S_j)\)：第\(j\)个指标的最大值

  2. 加权求和：按照预设权重计算综合相似度得分：

    \[ \text{综合得分}_i = \sum_{j=1}^{7} w_j \times S'_{ij} \]
    - \(w_j\)：第\(j\)个指标的权重
  3. 排序筛选：按综合得分从高到低排序，得分最高的年份即为最优相似年份。

2.5 模型优势

  • 全面性：同时考虑数值大小和变化趋势两个维度
  • 准确性：7个互补指标相互验证，避免单一指标的误判
  • 可解释性：每个指标都有明确的物理意义，便于工程师理解和验证
  • 稳定性：在小数据量场景下表现优异，不易过拟合

三、相似年修正径流预测模型

3.1 模型核心原理

本系统采用水文行业标准的相似年修正预测法，其基本逻辑是：如果两个年份的前期径流变化规律高度相似，那么它们的后期径流变化也会高度相似。

该方法是短期径流预测（当年剩余月份）中最成熟、最可靠、应用最广泛的方法，尤其适合只有月尺度历史径流数据的场景。

3.2 预测流程与公式

预测过程分为三个核心步骤：

步骤1：筛选最优相似年份

通过上述多指标加权综合相似度模型，从所有历史年份中筛选出与目标年份前期径流最相似的年份，记为\(Y_b\)。

步骤2：计算动态修正系数

使用目标年份已有实际数据的月份，计算目标年份与最优相似年份的平均比值，作为修正系数，消除不同年份之间的整体水量差异：

\[ k = \frac{1}{m}\sum_{i=1}^{m}\frac{X_{t,i}}{Y_{b,i}} \]
- \(k\)：动态修正系数
- \(m\)：目标年份已有实际数据的月份数（当前为4个月）
- \(X_{t,i}\)：目标年份第\(i\)个月的实际径流数据
- \(Y_{b,i}\)：最优相似年份第\(i\)个月的实际径流数据

步骤3：生成全年预测值

用修正系数乘以最优相似年份各月的径流数据，得到目标年份所有月份的预测值：

\[ \hat{X}_{t,j} = Y_{b,j} \times k \quad (j=1,2,...,12) \]
- \(\hat{X}_{t,j}\)：目标年份第\(j\)个月的预测径流数据
- \(Y_{b,j}\)：最优相似年份第\(j\)个月的实际径流数据

3.3 预测结果输出规则

系统严格按照以下规则输出结果：

  1. 原始径流数据列：100%保留原始数据，有数据显示数据，无数据为空值(NaN)
  2. 预测径流列：
    • 1936-2024年：所有月份全部为空值(NaN)
    • 2025年及以后：所有12个月份（无论有无实际数据）都填充预测值
  3. 相对误差列：仅计算目标年份有实际数据的月份的相对误差，其他月份为空值

3.4 模型精度说明

  • 前期拟合误差：最优相似年份与目标年份前期（1-4月）的平均相对误差约为10.5%
  • 全年预测误差：根据历史验证，相似年法的全年预测平均误差约为15%-20%
  • 汛期误差：6-9月汛期误差略高于枯水期，约为20%-25%（受降雨随机性影响较大）

3.5 与其他预测模型的对比

模型类型 适用场景 优点 缺点
相似年修正法 短期预测（当年）、数据有限场景 可解释性强、小数据表现好、计算高效 对极端事件预测能力有限
ARIMA/SARIMA 单变量线性时间序列 成熟稳定、理论基础扎实 只能捕捉线性关系
LSTM/GRU 长期复杂序列预测 能捕捉非线性关系 需要大量日尺度数据、可解释性差
物理水文模型 流域尺度高精度预测 物理机制明确 需要大量气象、地形等辅助数据

四、注意事项与后续优化方向

4.1 注意事项

  1. 数据更新：随着目标年份后续月份实际数据的陆续产生，系统会自动更新修正系数和预测值，预测精度会不断提高。建议每月更新一次数据并重新运行分析。

  2. 数据集维护：在DolphinScheduler中修改数据集结构时，建议先删除旧数据集再重新运行任务，避免列不匹配错误。每次代码修改导致输出列变化时，都应重新创建数据集。

  3. 极端事件：当出现极端洪水或干旱事件时，相似年法的预测精度会下降。此时建议结合气象预报数据、流域雨情信息进行人工修正，或采用多模型集成预测。

4.2 后续优化方向

  1. 多相似年份加权预测：不再仅使用最优相似年份，而是使用TOP3或TOP5相似年份进行加权平均，权重与相似度得分成正比，进一步提高预测稳定性和鲁棒性。

  2. 季节修正系数：针对不同季节（枯水期11-4月、汛期5-10月）使用不同的修正系数，重点提高汛期预测精度，因为汛期径流对水资源调度和防洪安全更为重要。

  3. 置信度评估：添加预测置信度列，基于相似年份的历史预测误差分布，量化每个预测值的可靠程度，为决策提供更全面的参考。

  4. 可视化输出：生成径流变化趋势图、相似度排名图、预测误差分布图等可视化图表，便于直观分析和汇报展示。

  5. 多因子融合预测：在数据允许的情况下，引入降雨量、气温、上游来水等辅助因子，构建相似年+机器学习混合模型，进一步提升预测精度。
